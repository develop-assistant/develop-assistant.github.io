Http参数签名

2020-06-13 tools tools, 验签 0 0 评论

前言

Http接口传输有很多加密方式, 如 AES 对称加密, RSA 非对称加密, 参数验签等, 下文主要简单示例参数如何验签。

原理

将 GET 参数和 appkey 通过如 MD5 之类的算法生成一个唯一不可逆的参数 sign, 参数传递的同时传递签名参数 sign。HTTP接收参数后根据自己持有的 appkey 和 appsecret 再次计算, 比对是否和传递的 sign 相同。

注意：appKey并不在http中传输，是计算摘要的密码盐
